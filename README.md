# Nominas

## Configuración Inicial

1. Copia appsettings.example.json y renómbralo a appsettings.json

2. Edita appsettings.json con tus datos reales:

- Información de la empresa

- Contraseñas de las bases de datos

- IPs de los servidores

⚠️ **IMPORTANTE**: appsettings.json está en .gitignore y NO se sube a GitHub por seguridad.
